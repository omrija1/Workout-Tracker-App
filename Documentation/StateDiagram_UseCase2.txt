
State Diagram for Use Case 2: Comprehensive Workout Session Creation

States:
- Start
- Select Workout Day
- Select/Create Exercise
- Log Sets and Reps
- Select Training Method
- Log Rest Time
- Log Weight
- Save and Continue
- End

Transitions:
- Start -> Select Workout Day: OpenApp()
- Select Workout Day -> Select/Create Exercise: ChooseDay()
- Select/Create Exercise -> Log Sets and Reps: ChooseExercise()
- Log Sets and Reps -> Select Training Method: InputSetsReps()
- Select Training Method -> Log Rest Time: ChooseMethod()
- Log Rest Time -> Log Weight: InputRestTime()
- Log Weight -> Save and Continue: InputWeight()
- Save and Continue -> Select/Create Exercise: SaveAndAddAnotherExercise()
- Save and Continue -> End: SaveAndFinish()

Error Handling:
- Any State -> Start: If error occurs, display error message and return to start

Conditions:
- User must complete each state to move to the next, except in the case of an error.
- User can choose to add another exercise after saving, which loops back to Select/Create Exercise state.
