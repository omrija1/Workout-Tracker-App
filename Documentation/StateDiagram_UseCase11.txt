
State Diagram for Use Case 11: User Profile and Settings

States:
1. Initial State (Authenticated)
2. Displaying Dashboard
3. Navigating to Settings
4. Fetching User Profile
5. Displaying Profile Form
6. Editing Profile Details
7. Updating User Profile
8. Displaying Update Result
9. Error State (Database error)

Transitions and Conditions:
1. OpenApp(): Initial State -> Displaying Dashboard
2. NavigateToSettings(): Displaying Dashboard -> Navigating to Settings
3. FetchUserProfile(): Navigating to Settings -> Fetching User Profile
4. DisplayProfileForm(profile_data): Fetching User Profile -> Displaying Profile Form
5. EditProfileDetails(new_details): Displaying Profile Form -> Editing Profile Details
6. UpdateUserProfile(new_details): Editing Profile Details -> Updating User Profile
7. UpdateResult(): Updating User Profile -> Displaying Update Result

Actions (Optional):
- Display an error message: If any database operation fails
